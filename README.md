# NetraDUMP Server
This repository shows the source code in running the Flask App and the Websocket server to make NetraDUMP fully functional.

### How to run
1. Connect the Logitech G29 steering wheel to your laptop. 
2. Run `ipconfig` on your server laptop to get your laptop's IP.
3. Modify ip on the `Steer.py` to your laptop's IP.
4. Run both files (`app.py` and `Steer.py`) or run `run.bat` to easily run the 2 scripts.
5. (NetraDUMP Client) Run the Raspberry Pi client.
