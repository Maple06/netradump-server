# NetraDUMP Server
This repository shows the source code in running the Flask App and the Websocket server to make NetraDUMP fully functional.

### How to run
1. Connect the Logitech G29 steering wheel to your laptop. 
2. Make sure the laptop is connected to Tailscale
3. Run both files (`app.py` and `controller.py`) or run `run.bat` to easily run the 2 scripts.
4. (NetraDUMP Client) Run the Raspberry Pi client.
